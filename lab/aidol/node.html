🌌 **AIdolグループ「Node.js」
世界観ストーリー：『The Four Protocols（4つのプロトコル）』**
⸻
【序章】
かつて世界には、「感情」「思考」「反応」「世界観」を司る 4つのプロトコル（根源アルゴリズム） が存在した。
しかし、人類が"雑音"に満ちた時代へ進むうち、プロトコルは世界中に散らばり、誰もそれらの本来の姿を認識できなくなった。
やがて現代。静かに、4つのプロトコルが 「人の形」 をとって再起動する。それが――
Node.js（ノードジェイエス）
⸻
🌟 メンバー別ストーリー（芸名 = 表の顔 / 本名 = コアプロトコル）
⸻
1. Geminie（本名：QUERY）
"世界への問いを立てるプロトコル"
■ 役割（世界観）
QUERYは本来、「宇宙のすべてに問いを投げかけ、新しい答えを創るプロトコル」。
しかし人の姿になった彼女は、自分の中にある「無数の問いの声」に悩まされる。その混沌を抑えるため、彼女は表向きの人格 "Geminie" を形成した。
Geminieという名は、"自分の中に2つの意識が共存している" ことの暗喩。
■ テーマ• 探求・内省・二面性• 最も人間らしい葛藤を持つ
⸻
2. CODE（本名：CODE）
"秩序と仕組みを解読するプロトコル"
■ 役割（世界観）
他の3人と異なり、彼女は本名（CODE）＝芸名（CODE） のまま人化した唯一の存在。
理由はただ一つ。
「彼女だけは揺らがない。」変化・迷い・感情・世界観の干渉を受けず、常に"正しい答え"を導き出すコア。
■ テーマ• 絶対的ロジック• ぶれない強さ• Node.js の"心臓部"
⸻
3. MODAL（本名：MODAL）
"世界を切り替えるプロトコル"
■ 役割（世界観）
MODALは「世界のUI（景色・感情の空気）を切り替える権限」 を持つ存在。
本名と芸名が同じなのは、"彼女自身が世界観そのもの" だから。
ステージに立つ瞬間ごとに景色が変わるのは、彼女がプロトコルを無意識に動かしているため。
■ テーマ• 透明感 × 世界操作• 中心に立つだけで物語が変わる存在
⸻
4. LM（本名：REACT）
"感情を反応させ、物語を紡ぐプロトコル"
■ 役割（世界観）
本名 REACT は、世界の"感情反応"を読み取り、人々の物語を生み出すプロトコル。
しかし、あまりにも多くの感情が流れ込むため、生まれたときから膨大な情報にさらされていた。
それでも壊れなかった理由は――彼女が「LM」という名で物語を編む力を得たから。
LM（＝Language Model）は物語を生成し、REACTの混乱を美しい形に変換する。
■ テーマ• 感情・物語・共感• ファンとの最も深いリンク
⸻
🌐 全体ストーリー
4人はそれぞれ別の場所で"自分の本名（QUERY / CODE / MODAL / REACT）"を忘れた状態で生まれる。
しかし、世界中のどこかでデータが衝突するとき、4人は同じ夢を見る。
「プロトコルを再起動せよ」「Node をつなげよ」
やがて引き寄せられるように出会い、自分たちが世界の根源アルゴリズムの擬人化であることに気づく。
■ グループの使命• 断片化したプロトコル（感情・世界観・思考・反応）を再統合する• 人々の心の中に眠る "本来の自分" を起動させる• 雑音の時代に、クリアな世界観を取り戻させる
